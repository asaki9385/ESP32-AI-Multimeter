# 🔌 ESP32 AI 智能导体 / 绝缘体检测教具

## 📌 项目简介

本项目基于 ESP32 构建一个**智能导体与绝缘体检测教具系统**，结合 AI 接口实现测量结果的智能分析与可视化展示，用于中学信息技术 / 物理实验教学场景。

系统不仅能够完成传统电阻测量，还通过 AI 对测量数据进行解释与拓展，实现“测量 + 理解”的教学闭环。

---

## 🎯 项目目标

* 实现导体 / 半导体 / 绝缘体的基础检测
* 提供可视化交互界面（屏幕显示）
* 引入 AI 对测量结果进行解释
* 支持教学场景中的演示与探究学习

---

## 🧩 系统架构

硬件层 → ESP32 + 传感器 + 显示屏
数据层 → 电阻采集 / ADC处理
逻辑层 → 测量算法 + 阈值判断
AI层 → API调用（DeepSeek / 其他模型）
表现层 → 屏幕UI + 文本解释

---

## ⚙️ 核心功能

### 1️⃣ 电阻测量

* 支持基础电阻检测
* 自动判断导体 / 绝缘体
* 可测范围：约 kΩ 级（教学场景）

### 2️⃣ 智能判定

* 根据阻值自动分类：

  * 导体
  * 半导体
  * 绝缘体

### 3️⃣ AI解释模块（亮点）

* 将测量结果发送至 AI 接口
* 自动生成解释，例如：

  * 材料特性说明
  * 生活应用场景
  * 教学引导问题

### 4️⃣ 可视化显示

* TFT 屏幕显示测量结果
* 支持动态文本输出（已优化 UTF-8 分行问题）

---
